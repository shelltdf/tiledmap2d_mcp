# 用户说明书：TiledMap2D

## 环境

- 现代浏览器（支持 ES 模块与 Canvas）。

## 启动

- 开发：在 `tiledmap2d` 目录执行 `npm install` 后 `npm run dev`，按终端提示打开本地 URL。
- 预览构建结果：`npm run build` 后 `npm run preview`。

## 使用

1. **主菜单**：标题栏下方为 **文件 / 编辑 / 帮助** 菜单；**帮助 → 支持的文件格式** 可查看 JSON 与 TMX 说明。
2. **新建**：在工具栏设置宽度、高度，点击「新建地图」。
3. **绘制**：在左侧调色板选择瓦片类型，在画布上按住左键拖拽绘制。
4. **擦除**：选择「空」瓦片或右键（若实现支持）。
5. **缩放**：在画布上滚动鼠标滚轮。
6. **平移**：中键拖拽画布区域（或拖动滚动条）。
7. **导出 JSON**：菜单或工具栏「导出 JSON」下载 `.json` 地图文件。
8. **导入 JSON**：菜单或工具栏「导入 JSON」选择此前导出的 `.json` 文件。
9. **导出 TMX**：菜单或工具栏「导出 TMX」下载可在 **Tiled** 中打开的正交地图（建议将同目录下的 `tilemap-palette.png` 一并放到 `.tmx` 旁，以便图块显示正常）。
10. **导入 TMX**：菜单或工具栏「导入 TMX」选择 `.tmx`（支持 Tiled 默认的 CSV 或 zlib 压缩图层）。

## 常见问题

- **导入失败**：确认文件为 UTF-8 JSON，且字段与版本一致（见开发文档中的规格）。
