# 用户说明书：TiledMap2D

## 环境

- 现代浏览器（支持 ES 模块与 Canvas）。

## 启动

- 开发：在 `tiledmap2d` 目录执行 `npm install` 后 `npm run dev`，按终端提示打开本地 URL。
- 预览构建结果：`npm run build` 后 `npm run preview`。

## 主窗口布局（Windows 风格）

自上而下：**标题** → **菜单栏**（文件 / 编辑 / 帮助）→ **工具栏** → **主工作区** → **状态栏**。

- **左侧 Dock「地图层」**：折叠后「地图层」按钮在**最靠左缘**窄列，与中间面板区并排；可点标题栏 **⟨** 折叠（再点缘条展开）；图层列表与添加、改名、删除（至少保留一层）；拖拽排序；点击行切换当前编辑层；勾选显隐。
- **中央客户区**：画布；画布内左上为 **工具栏**（选择 / 画笔 / 橡皮 / 适应 / 居中），右上为 **显示**（三个复选框：网格、原点、参考坐标）。
- **右侧**：上为 **块库**，下为 **块定义**（各为一个停靠区）；折叠后按钮在**最靠右缘**的窄列，与中间面板区为两列并排；二者均可点标题栏 **⟩** 折叠（再点缘条上对应名称展开）。

## 使用

1. **新建地图**：工具栏 **新建地图…**，在对话框中设置宽度、高度、瓦片像素边长（格总数会显示）；确认后创建地图。
2. **地图设置**：工具栏 **地图设置…**，可修改当前地图的宽高与瓦片边长（地图格数据会按规则裁剪或扩展）。
3. **主菜单**：**帮助 → 支持的文件格式** 可查看 JSON 与 TMX 说明。
4. **图层**：见左侧「地图层」；**添加** 打开对话框选择层类型（瓦片/图片）与插入位置。
5. **块库**：选择瓦片类型；可折叠/展开（见上）；顶部 **导入**（JSON 块库）、**编辑**、**删除**（「空」类型不可删）。
6. **绘制**：中央画布按住左键拖拽（仅改当前**瓦片**层）。
7. **块定义**：右侧展示当前选中块说明；可折叠/展开（与块库相同）。
8. **擦除**：选「空」块或右键擦除。
9. **缩放**：在画布区域滚动滚轮（以指针为锚点）。
10. **平移**：在画布区域 **中键拖拽**；画布聚焦时 **方向键** 平移视口。
11. **块库中键**：在块库某项上 **中键** 可快速选中该块。
12. **工具栏（画布内）**：**选择** — 左键点格拾取该格块类型并高亮；**画笔** / **橡皮**；**适应** — 缩放使地图大致充满视口并居中；**居中** — 仅滚动居中。
13. **空格**：画布聚焦时按住 **空格** 临时变为选择工具，松手恢复。
14. **显示**：勾选网格、原点、参考坐标等辅助显示。
15. **导出 JSON**：菜单或工具栏（多图层，**version 3**；仍兼容导入 v1/v2）。
16. **导入 JSON** / **导出 TMX** / **导入 TMX**：菜单或工具栏；TMX 在 **Tiled** 中使用时可配合仓库内示例 `tilemap-palette.png`。

## 常见问题

- **导入失败**：确认 UTF-8 JSON/TMX，字段与版本一致；瓦片 id 不得超过当前块库类型数量减一。
- **TMX 与块数不一致**：导出 TMX 的 tileset 占位为 5 列示例图；若块库类型数量变化，在 Tiled 中可能需自配图块资源。
