# 用户说明书：瓦片地图编辑器

## 环境

- 现代浏览器（支持 ES 模块与 Canvas）。

## 启动

- 开发：在 `tilemap-editor-vue` 目录执行 `npm install` 后 `npm run dev`，按终端提示打开本地 URL。
- 预览构建结果：`npm run build` 后 `npm run preview`。

## 使用

1. **新建**：在工具栏设置宽度、高度，点击「新建地图」。
2. **绘制**：在左侧调色板选择瓦片类型，在画布上按住左键拖拽绘制。
3. **擦除**：选择「空」瓦片或右键（若实现支持）。
4. **缩放**：在画布上滚动鼠标滚轮。
5. **平移**：中键拖拽画布区域（或拖动滚动条）。
6. **导出**：「导出 JSON」下载地图文件。
7. **导入**：「导入 JSON」选择此前导出的 `.json` 文件。

## 常见问题

- **导入失败**：确认文件为 UTF-8 JSON，且字段与版本一致（见开发文档中的规格）。
