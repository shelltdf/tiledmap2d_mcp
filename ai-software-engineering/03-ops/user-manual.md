# 用户说明书：TiledMap2D

## 环境

- 现代浏览器（支持 ES 模块与 Canvas）。

## 启动

- 开发：在 `tiledmap2d` 目录执行 `npm install` 后 `npm run dev`，按终端提示打开本地 URL。
- 预览构建结果：`npm run build` 后 `npm run preview`。

## 使用

1. **主菜单**：标题栏下方为 **文件 / 编辑 / 帮助** 菜单；**帮助 → 支持的文件格式** 可查看 JSON 与 TMX 说明。
2. **新建**：在工具栏设置宽度、高度，点击「新建地图」。
3. **图层**：左侧「地图层」顶部 **添加 / 改名 / 删除** 作用于 **当前选中层**（至少保留一层）；列表 **点击行** 切换当前编辑层（绘制、擦除、右键等仅改此层）；勾选显隐；改名在弹窗中输入（默认第一层为「地面」）。
4. **绘制**：在右侧「块库」选择瓦片类型；中央画布按住左键拖拽绘制（仅改当前层）。
5. **块定义**：右侧展示当前笔刷对应块的内建说明（只读）。
6. **擦除**：选择「空」块或右键擦除。
7. **缩放**：在画布上滚动鼠标滚轮。
8. **平移**：**中键拖拽**画布区域；或拖动滚动条；画布聚焦时 **方向键** 可平移视口滚动区。
9. **块库中键**：在右侧块库某项上 **中键点击** 可快速选中该块（与左键相同效果）。
10. **工具栏**：画布内左侧：选择 / 画笔 / 橡皮（图标）。画笔为默认；橡皮模式下左键等效擦除；**选择** 模式下左键点格子会 **高亮该格** 并在 **块库** 中选中该格在当前层的块类型（不绘制）。
11. **导出 JSON**：菜单或工具栏「导出 JSON」下载 `.json`（多图层，版本 2；仍可导入旧版版本 1 单层文件）。
12. **导入 JSON**：菜单或工具栏「导入 JSON」选择 `.json` 文件。
13. **导出 TMX**：菜单或工具栏「导出 TMX」下载可在 **Tiled** 中打开的正交地图（多图层；建议将同目录下的 `tilemap-palette.png` 一并放到 `.tmx` 旁）。
14. **导入 TMX**：菜单或工具栏「导入 TMX」选择 `.tmx`（多图层；支持 CSV 或 zlib 压缩图层数据）。

## 常见问题

- **导入失败**：确认文件为 UTF-8 JSON，且字段与版本一致（见开发文档中的规格）。
