# 详细设计：TiledMap2D

## 坐标系

- **格坐标**：`(gx, gy)`，整数，`0 <= gx < width`，`0 <= gy < height`。
- **屏幕到格**：考虑画布在视口内的位置、`devicePixelRatio`、缩放 `zoom`、平移 `panX`/`panY`，将指针坐标转为格索引；越界不绘制。

## 绘制交互（工具）

- **选择工具**：左键点选、Ctrl+左键多选、拖框、Shift+拖框追加；右键清空选区；仅高亮选区，**不**修改块库当前块。
- **吸取工具**：左键读取**当前编辑层**该格 tile id，写回块库当前笔刷；**任意工具下按住 Alt** 可临时当作吸取（与 **空格** 临时选择互斥；输入框焦点时无效）。
- **填充工具**：用当前块类型写入格；**无选区**时单击或**拖动**连续作用于路径上的格；**有选区**时须**点击落在选区内**才对整段选区填充，点在选区外无效。
- **橡皮工具**：逻辑同填充，目标为「空」tile。
- **空格**：按住临时切换为选择工具，松开恢复先前工具。

## 主菜单与帮助

- **主菜单**：标题栏下为 Windows 风格菜单（**文件 / 编辑 / 语言 / 主题 / 帮助**）；**帮助 → 支持的文件格式** 打开模态对话框；**语言** 切换界面语言；**主题** 为跟随系统 / 浅色 / 深色。

## 视口与画布内 UI

- **缩放**：滚轮改变 `zoom`，限制在 `minZoom`～`maxZoom`，以指针为锚点。
- **平移**：画布内容通过 **`viewport-inner` 的 CSS `translate`（panX/panY）** 平移；**中键拖拽**更新平移量；画布聚焦时 **方向键** 步进平移。
- **适应 / 居中**：工具栏按钮调整缩放并居中内容。
- **显示** 面板：复选框控制背景网格、原点标记、碰撞体积等；叠在视口右上，不随画布平移滚动。

## 状态栏与 Log

- **状态栏**展示就绪/错误摘要、光标格、缩放；**点击状态栏** 打开 **Log** 对话框，可查看 info/warn/error 历史并复制全部。

## 图层

- 渲染顺序为 `layers` 数组顺序，仅绘制 `visible` 为真的层；上空格透出下层。
- 编辑目标为当前 `activeLayerIndex` 对应层；至少保留一层；`kind` 含 **tile / image / area**（区域层与瓦片层同属可绘制可拾取逻辑，见 `spec.md`）。

## Dock 与布局

- **地图层**：左侧 Dock；添加/改名/删除、拖拽排序、显隐。
- **块库**、**块定义**：右侧一列，各占一个 Dock；二者与地图层 Dock 类似可折叠为缘条按钮；块库含导入/编辑/删除与类型列表。

## 导入导出

- **JSON**：当前导出 **version 3**；兼容读入 v1/v2/v3。失败时状态栏与 Log 展示错误信息（见 `02-physical/tiledmap2d/spec.md`）；用户可见文案随界面语言变化。
- **TMX**：导出多 `<layer>`；导入按顺序解析 tile layer；GID 按 `spec` 转为编辑器瓦片 id（块类型数量由当前 `tileTypes.length` 约束）。
