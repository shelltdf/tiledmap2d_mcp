# 详细设计：TiledMap2D

## 坐标系

- **格坐标**：`(gx, gy)`，整数，`0 <= gx < width`，`0 <= gy < height`。
- **屏幕到格**：考虑画布在视口内的位置、`devicePixelRatio`、缩放 `zoom`，将指针坐标转为格索引；越界不绘制。

## 绘制交互

- **左键按下并拖拽**：连续绘制当前选中瓦片 id（**瓦片层**；图片层不绘制）。
- **右键**：擦除（id=0）。
- **选择工具**：左键拾取当前瓦片层该格块 id，高亮格并同步块库选中 id。
- **空格**：按住临时切换为选择工具，松开恢复先前工具。

## 主菜单与帮助

- **主菜单**：标题栏下为 Windows 风格菜单（文件 / 编辑 / 帮助）；**帮助 → 支持的文件格式** 打开模态对话框。

## 视口与画布内 UI

- **缩放**：滚轮改变 `zoom`，限制在 `minZoom`～`maxZoom`，以指针为锚点调整滚动位置。
- **平移**：`viewport-scroll` 上 `overflow: auto`；**中键拖拽**更新 `scrollLeft`/`scrollTop`（指针在 **viewport-inner** 上捕获，含留白）；画布聚焦时 **方向键** 步进滚动。
- **适应 / 居中**：工具栏按钮调整缩放并滚动至居中。
- **显示** 面板：复选框控制背景网格、原点标记、参考坐标文字；叠在视口右上，不随画布滚动。

## 图层

- 渲染顺序为 `layers` 数组顺序，仅绘制 `visible` 为真的层；上空格透出下层。
- 编辑目标为当前 `activeLayerIndex` 对应层；至少保留一层。

## Dock 与布局

- **地图层**：左侧 Dock；添加/改名/删除、拖拽排序、显隐。
- **块库**、**块定义**：右侧一列，各占一个 Dock；二者与地图层 Dock 类似可折叠为缘条按钮；块库含导入/编辑/删除与类型列表。

## 导入导出

- **JSON**：当前导出 **version 3**；兼容读入 v1/v2/v3。失败时状态栏错误信息（见 `02-physical/tiledmap2d/spec.md`）。
- **TMX**：导出多 `<layer>`；导入按顺序解析 tile layer；GID 按 `spec` 转为编辑器瓦片 id（块类型数量由当前 `tileTypes.length` 约束）。
