# 产品设计：TiledMap2D（2D 正交瓦片地图编辑器）

## 命名与边界

- 仓库/实现标识 **TiledMap2D**（`tiledmap2d`）：指 **二维、正交网格上的瓦片（tile）数据**，用于关卡、地形稿等。
- **不包含**：GIS / 地球空间参考下的 **地图瓦片**（如 Web Mercator、经纬度切片、球面金字塔）——与本产品无关。

## 愿景

提供在浏览器中运行的 **2D 正交瓦片地图编辑器**，视觉与交互贴近 **Windows 11 浅色风格**（Segoe UI、工具栏、状态栏、系统色强调色），面向关卡/地形稿的快速绘制与保存。

## 目标用户与场景

- 需要快速铺砖、试玩关卡布局的开发者或策划。
- 本地或内网使用，通过 JSON 与工具链对接。

## 范围（MVP）

- 可配置地图宽高（瓦片格数）、瓦片像素尺寸。
- **多图层**：默认「地面」层；可增删层、显隐、重命名与选择当前编辑层；画布按层自下而上叠加绘制。
- **块库**（原调色板）选择瓦片类型；**块定义**区展示当前选中类型的内建说明；左键绘制、右键擦除。
- 缩放（滚轮）、平移（滚动条；画布聚焦时 **方向键** 平移视口）。
- 新建、清空、导出/导入 JSON（v1/v2）与 TMX（多图层）。
- 状态栏显示光标格坐标、当前图层名与缩放。

## 非目标（当前版本）

- 用户自定义块类型（颜色/名称编辑）、图块集图片导入、碰撞层、自动平铺规则。

## UI 几何说明

主界面自上而下：**标题区 / 工具栏 / 主工作区（左侧地图层 dock + 中央画布 + 右侧块库与块定义 dock）/ 状态栏**。详细控件布局见 `01-logic` 与实现。
