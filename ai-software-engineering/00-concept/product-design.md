# 产品设计：2D 瓦片地图编辑器

## 愿景

提供在浏览器中运行的 **2D 瓦片地图编辑器**，视觉与交互贴近 **Windows 11 浅色风格**（Segoe UI、工具栏、状态栏、系统色强调色），面向关卡/地形稿的快速绘制与保存。

## 目标用户与场景

- 需要快速铺砖、试玩关卡布局的开发者或策划。
- 本地或内网使用，通过 JSON 与工具链对接。

## 范围（MVP）

- 可配置地图宽高（瓦片格数）、瓦片像素尺寸。
- 单层瓦片网格绘制；调色板选择瓦片类型；左键绘制。
- 缩放（滚轮）、平移（中键拖拽或滚动条）。
- 新建、清空、导出 JSON、从 JSON 导入。
- 状态栏显示光标格坐标与当前缩放。

## 非目标（当前版本）

- 多图层编辑、图块集图片导入、碰撞层、自动平铺规则。

## UI 几何说明

主界面自上而下：**标题区 / 工具栏 / 主工作区（左侧调色板 + 中央画布）/ 状态栏**。详细控件布局见 `01-logic` 与实现。
