# 产品设计：TiledMap2D（2D 正交瓦片地图编辑器）

## 命名与边界

- 仓库/实现标识 **TiledMap2D**（`tiledmap2d`）：指 **二维、正交网格上的瓦片（tile）数据**，用于关卡、地形稿等。
- **不包含**：GIS / 地球空间参考下的 **地图瓦片**（如 Web Mercator、经纬度切片、球面金字塔）——与本产品无关。

## 愿景

提供在浏览器中运行的 **2D 正交瓦片地图编辑器**，视觉与交互贴近 **Windows 11 浅色风格**（Segoe UI、工具栏、状态栏、系统色强调色），面向关卡/地形稿的快速绘制与保存。

## 目标用户与场景

- 需要快速铺砖、试玩关卡布局的开发者或策划。
- 本地或内网使用，通过 JSON 与工具链对接。

## 壳层类型（SDI）

- 采用 **SDI（单文档界面）** 心智模型：**单页应用、单一主窗口**；一次编辑会话聚焦当前地图数据；多地图通过「新建/导入」切换当前文档（非多顶层 OS 窗口）。
- 主框架遵循 **Windows 风格纵向分区**（见 `.cursor/rules/window-gui-documentation.mdc`）：**菜单栏** → **工具栏** → **主工作区** → **状态栏**；中间 **客户区** 为画布；左右为 **Dock**。

## 范围（MVP）

- 可配置地图宽高（瓦片格数）、瓦片像素尺寸（**新建地图…** / **地图设置…** 对话框）。
- **多图层**：默认「地面」层；可增删层、显隐、重命名与选择当前编辑层；画布按层自下而上叠加绘制。
- **Dock（停靠容器）**：**地图层**（左）、**块库**（右上一）、**块定义**（右下一）各为 **独立 Dock**（见窗口 GUI 规则）；块库可选择瓦片类型；块定义区展示当前选中类型；左键绘制、右键擦除。
- **块库**：支持 **导入 / 编辑 / 删除** 瓦片类型（JSON）；与地图格数据联动（裁剪/重排规则见物理规格）。
- 缩放（滚轮，以指针为锚点）、平移（中键拖拽、方向键；视口内 **适应 / 居中**）；画布内 **显示** 浮层：网格/原点/参考坐标（复选框）。
- 画布内 **工具栏**（工具栏标题 + 选择/**吸取**/填充/橡皮 + 清空地图 + 适应/1:1）；**选择**仅框选与高亮，不改块库；**吸取**从地图格同步块类型到块库；**填充**对选区批量铺块，无选区时仅填充一格；按住 **空格** 临时切换为选择工具，松手恢复。
- 新建、清空、导出/导入 JSON（v1/v2/**v3**）与 TMX（多图层）。
- 状态栏显示光标格坐标、当前图层名与缩放。

## 非目标（当前版本）

- 图块集图片导入、碰撞层、自动平铺规则（块库颜色为纯色占位，非图集贴图编辑）。

## UI 几何说明

主界面自上而下：

| 区域 | 内容 |
|------|------|
| 标题区 | 应用名与副标题 |
| **菜单栏** | 文件 / 编辑 / 语言 / 主题 / 帮助 |
| **工具栏** | 新建地图…、地图设置…、清空、JSON/TMX 导入导出 |
| **主工作区** | **左 Dock「地图层」** \| **中央客户区（画布 + 画布内工具栏与「显示」面板）** \| **右 Dock 列：块库、块定义** |
| **状态栏** | 提示 / 光标 / 缩放；可点击打开 **Log** |

中央客户区内：**画布**为滚动区域；**画布工具栏**叠在视口内左上；**显示**（网格/原点/参考）叠在视口内右上，与工具栏同属覆盖层、不随中键平移滚动。

详细交互见 `01-logic/detailed-design-tiledmap2d.md` 与 `02-physical/tiledmap2d/spec.md`。
