# 软件设计

## 子系统

1. **壳层（Windows 风格 SDI）**：标题、**菜单栏**、**工具栏**、**主工作区**（多 **Dock** + 中央 **客户区**）、**状态栏**；主题 CSS 变量（`win-theme.css`）。
2. **地图模型**：多层二维瓦片网格；每层含名称、显隐、`kind`（瓦片/图片占位）与 `tiles`；元数据（宽高、tileSize、JSON 版本）。
3. **编辑交互**：画布命中测试、对**当前层**绘制、缩放；视口滚动与中键平移；**空格**临时选择工具。
4. **块库 Dock**：可变 **tileTypes** 列表（导入/编辑/删除）；当前笔刷 id；与 **块定义 Dock** 联动展示选中类型。
5. **地图层 Dock**：增删层、显隐、重命名、拖拽排序、选择编辑层；添加图层对话框（瓦片/图片、插入位置）。
6. **画布内 UI**：工具栏（选择/填充/橡皮/清空/适应/1:1）、**显示** 面板（复选框）。
7. **地图设置**：新建/编辑地图尺寸与 tileSize（对话框）。
8. **持久化**：JSON v3 与 TMX 多图层序列化；浏览器内导入导出。

## 技术栈

- Vue 3（Composition API）、Vite。
- 画布渲染采用 HTML5 Canvas 2D。
