# 软件设计

## 子系统

1. **壳层（Windows 风格）**：布局、主题 CSS 变量、工具栏与状态栏。
2. **地图模型**：多层二维瓦片网格；每层含名称、显隐与 `tiles`；元数据（宽高、tileSize、JSON 版本 v1/v2）。
3. **编辑交互**：画布命中测试、对**当前层**绘制、缩放；视口滚动与方向键平移。
4. **块库与块定义**：内建瓦片类型列表、当前笔刷；块定义区只读展示选中类型。
5. **图层 dock**：增删层、显隐、重命名、选择编辑层。
6. **持久化**：JSON 与 TMX 多图层序列化；浏览器内导入导出。

## 技术栈

- Vue 3（Composition API）、Vite。
- 画布渲染采用 HTML5 Canvas 2D。
